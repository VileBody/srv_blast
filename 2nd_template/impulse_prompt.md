Тебе дан аудиофайл. Выполни три задачи последовательно.

## 1. Извлечь тайминги слов

```json
{ "word_timings": [
    { "word": "слово", "start": 0.000, "end": 0.300 },
    ...
]}
```

- Точность ±0.05с
- Каждое слово отдельно, включая короткие ("и", "в", "но")

---

## 2. Разметить субтитры

**Типы строк:**

**long** — основная строчка. Никогда не содержит акцентное слово если оно идёт следом как short.
**short** — акцентное слово или неразрывная двухсловная фраза-рефрен. Только императивы и слова с эмоциональным ударом: "прости", "уйди", "навсегда", "конец", "сотри", "ride it". Описательные слова ("горело", "внутри", "чужими") — не short. Если рефрен трека двухсловный ("ride it", "не сошлись") — short всегда содержит оба слова вместе.

**Тайминги — единое правило для всех слоёв:**
- `in` = `start` этого слова (или первого слова строчки) из word_timings
- `out` = `start` первого слова следующего слоя в JSON — НЕ `end` последнего слова
- Последний слой: `out` = `end` последнего слова + 0.5с

**Два варианта short:**

**А — standalone** (слово само по себе, без long перед ним):
```json
{ "text": "прости", "in": 0.000, "out": 1.240, "type": "short" }
```
`out` = `start` следующего слоя (как у всех).

**Б — после long** (long заканчивается где начинается short):
```json
{ "text": "мы станем", "in": 1.240, "out": 1.900, "type": "long"  },
{ "text": "чужими",   "in": 1.900, "out": 2.390, "type": "short" }
```
`out` long = `in` short. Long в JSON всегда стоит перед своим short.

**Когда NOT делать short:**
- Слово звучит < 0.4с
- После short следующий слой начинается менее чем через 0.4с — слово остаётся в long
- Уже был short в предыдущей строчке (максимум 1 short на 2 строчки)
- Исключение из лимита: рефрен (одно слово повторяется подряд 3+ раз) — каждое повторение short

**Разбивка:**
- Максимум 15-18 символов на строчку
- Минимум 0.6с для long — если после отрезания акцентного слова long получается < 0.6с, не делай short, оставь слово в long
- Не обрывай строчку на предлоге или союзе ("то что", "всё то", "как", "но")

**Регистр и пунктуация:**
- Только строчные буквы
- Убирай: , . ? ! ; : — – ... « » " ( ) /
- Оставляй: ' апостроф внутри слова (i'm, we're), - дефис внутри слова (как-то, когда-то)

## 3. Нормализовать тайминги

`новый_тайминг = старый_тайминг - in_первого_слоя`

Верни word_timings, затем итоговый JSON.
