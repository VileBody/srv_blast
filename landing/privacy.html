Политика конфиденциальности

Дата публикации: 27 марта 2026 г.

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сервиса Blast (далее — «Сервис»), принадлежащего ИП Чернов Никита Романович, ИНН 623013205426, ОГРНИП 324620000005644 (далее — «Оператор»).

1. Собираемые данные

1.1. При использовании Сервиса Оператор может собирать следующие данные:

  • Идентификатор пользователя в Telegram;
  • Аудиоматериалы и текстовые данные, предоставленные пользователем для генерации контента;
  • Сведения об оплате (без хранения данных банковских карт — обработка платежей осуществляется АО «Т-Банк»);
  • Техническая информация (IP-адрес, тип устройства, данные о браузере) при посещении веб-сайта.

2. Цели обработки данных

2.1. Персональные данные обрабатываются исключительно для:

  • Оказания услуг по генерации видеоконтента;
  • Обработки платежей;
  • Связи с пользователем по вопросам оказания услуг;
  • Улучшения качества Сервиса.

3. Защита данных

3.1. Оператор принимает необходимые организационные и технические меры для защиты персональных данных от несанкционированного доступа, изменения, раскрытия или уничтожения.

3.2. Передача данных между устройством пользователя и серверами Сервиса осуществляется по зашифрованному протоколу HTTPS/TLS, что обеспечивает конфиденциальность информации в процессе передачи.

3.3. Данные банковских карт не хранятся и не обрабатываются на серверах Оператора. Приём и обработка платежей осуществляется АО «Т-Банк» (tbank.ru) с использованием протокола шифрования TLS и сертификации стандарта PCI DSS.

4. Безопасная передача платёжных данных

4.1. При совершении оплаты пользователь перенаправляется на защищённую платёжную страницу АО «Т-Банк». Все платёжные данные (номер карты, срок действия, CVV) вводятся исключительно на стороне банка и не передаются Оператору.

4.2. Платёжный шлюз АО «Т-Банк» соответствует международному стандарту безопасности PCI DSS Level 1, что гарантирует максимальный уровень защиты конфиденциальной информации плательщиков.

4.3. Оператор не запрашивает и не хранит данные банковских карт пользователей.

5. Передача данных третьим лицам

5.1. Оператор не передаёт персональные данные пользователей третьим лицам, за исключением случаев:

  • Обработки платежей через АО «Т-Банк»;
  • Требований, предусмотренных законодательством Российской Федерации.

6. Хранение данных

6.1. Персональные данные хранятся в течение срока, необходимого для достижения целей их обработки, но не менее срока, установленного действующим законодательством.

6.2. Пользователь вправе запросить удаление своих персональных данных, направив запрос на адрес: support@blast808.com.

7. Файлы cookie

7.1. Веб-сайт Сервиса может использовать файлы cookie для обеспечения корректной работы и анализа посещаемости. Пользователь может отключить использование cookie в настройках браузера.
