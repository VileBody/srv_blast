# Fonts

Place the following `.woff2` files in this directory before deploying:

| File | Weight | Style |
|------|--------|-------|
| `Point-Regular.woff2` | 400 | normal |
| `Point-Medium.woff2`  | 500 | normal |
| `Point-Bold.woff2`    | 700 | normal |
| `Point-Italic.woff2`  | 400 | italic |
| `DriveHearts.woff2`   | 400 | normal |

**Point** — export from Figma: File → select text → right panel → font name → export via Figma desktop app
**DriveHearts** — decorative font for CTA background text only
